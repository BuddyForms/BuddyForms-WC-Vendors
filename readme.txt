BuddyPress WooCommerce Vendors Integrates WC Vendors with BuddyPress and enables a BuddyPress Social Marketplace

== Description ==

With BP WC Vendors you are able to build a highly customisable BuddyPress driven Marketplace. All WC Vendor Dashboard Tabs get integrated into the BuddyPress Member Profiles.

We have build many BuddyPress driven shops and marketplaces and put all our experiances and knowledge into this solution-
We belive with the right combination of plugins BuddyPress is the best choice to build a powerfull marketplaces.
Our focus is to have a extendable and flexible envirement ready to build up the marketplace you need.

<h3>Plugin Features</h3>
<h4>Deactivate BP Vendor Daschboard Tabs</h4>
You can deactivate Tabs if you not need them in your setup
<ul>
<li>Turn off "Products" tab</li>
<li>Turn off "Orders" tab</li>
<li>Turn off "Settings" tab</li>
<li>Turn off "Ratings" tab</li>
<li>Turn off "Coupongs" tab</li>
</ul>
<br><br>

<h4>Deactivate WordPress Dashboard for Vendors</h4>
By default, only vendors will be redirected to their BuddyPress 'Member Profile Vendors Dashboard' if they try to access the backend ( /wp-admin ). All other roles will be able to access the wp admin.
In the WC Vendor Pro settings you can set WordPress Dashboard to "Only administrators can access the /wp-admin/ dashboard".

== Installation ==

You can download and install the plugin using the built in WordPress plugin installer. If you download the plugin manually,
make sure it is uploaded to "/wp-content/plugins/".

Activate the plugin in the "Plugins" admin panel using the "Activate" link.
